## ðŸ”‘ Key Features

### AI-Powered Editing
- Multi-provider LLM support (15+ providers including OpenAI, Anthropic, DeepSeek, Ollama, and more)
- Context-aware code suggestions
- Fast and slow diff-based code replacement
- Support for reasoning-enabled models

### Real-Time Collaboration
- P2P collaboration without central servers
- Live cursor and presence tracking
- Synchronized file editing and terminal access
- Cross-peer chat synchronization

### Advanced Editing
- Multi-cursor support for efficient editing
- Cursor tracking and synchronization
- Visual diff preview before applying changes
- Smart code completion and refactoring

### Flexible Configuration
- Model selection per feature (chat, quick edit, search)
- Provider configuration with API keys
- Keyboard shortcuts customization
- Feature toggles and preferences
